# VISUAL-FRAMEWORK.md — 夜信视觉框架

> 版本 1.0 | 2026-06-22

---

## 1. 设计理念

**"夜空中的信笺"** — 深蓝夜空为底，暖金星光为引，营造安静、温暖、有仪式感的记录氛围。

核心视觉关键词：沉静、柔和、呼吸感、微光。

---

## 4. 间距与圆角

### 间距基准
- 页面水平内边距：28-32px
- 组件间距：8-16px
- 区块间距：24-48px

### 圆角
- 小按钮/标签：12-14px
- 卡片：16-20px
- 输入框：14-16px
- 圆形按钮：50%
- 大卡片/浮层：24px

## 6. 动画规范

### 缓动函数

| 名称 | 值 | 用途 |
|------|-----|------|
| ease | `cubic-bezier(.4,0,.2,1)` | 通用过渡 |
| spring | `cubic-bezier(.34,1.56,.64,1)` | 弹性效果 |
| bounce | `cubic-bezier(.68,-.55,.265,1.55)` | 弹跳效果 |

### 时长

| 场景 | 时长 |
|------|------|
| 微交互（hover/active） | 0.15-0.2s |
| 页面切换 | 0.4-0.5s |
| 入场动画 | 0.8-1.8s |
| 持续动画（极光） | 6s+ 循环 |

### 性能约束
- 极光动画：3 色块 + 12 粒子（已优化，低性能消耗）
- 优先使用 `transform` 和 `opacity`，避免触发重排
- 滚动容器使用 `-webkit-overflow-scrolling: touch`

## 7. 页面布局

### 视口
- 设计基准：390px 宽（iPhone 12/13/14）
- 最大宽度：430px，居中显示
- 比例标记：9:16（`<meta name="canvas:aspect-ratio" content="9:16">`）

### 页面结构
```
┌──────────────────────┐
│    Canvas 极光背景     │ ← 仅首页
├──────────────────────┤
│    页面标题/导航头     │ ← 固定顶部
├──────────────────────┤
│                      │
│    内容区域（可滚动）   │ ← flex:1, overflow-y:auto
│                      │
├──────────────────────┤
│    底部导航栏         │ ← 固定底部，3 个图标
└──────────────────────┘
```

### 导航
- 底部 3 个主入口：首页 / 写信 / 广场
- 顶部导航栏返回按钮 + 页面标题
- 页面切换动画：右滑入（0.5s ease）
