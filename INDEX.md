##  快速开始

### 查看原型
1. 双击打开 `night-letter.html`
2. 或右键 → 用浏览器打开
3. 推荐使用 Chrome / Edge / Safari

### 核心交互流程
```
首页星空 → 点击"写一封信" → 输入梦境 → 点击"寄出" 
→ 等待寄送动画(2.8s) → 阅读回信 → 点击"收好"返回首页
```

### 其他页面
- **星图页**：点击底部导航栏"星图"图标，可点击星星查看详情
- **发现页**：点击底部导航栏"发现"图标，查看模式分析

## 🔧 技术要点

### Canvas 性能参数
- 星星总数：200颗（远层120 + 中层60 + 近层20）
- 星云数量：4团
- 漂浮粒子：20个
- 流星频率：每3秒随机生成

### 响应式断点
- 移动端优先：375-414px 宽度优化
- 支持触摸视差交互
