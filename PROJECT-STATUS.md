# PROJECT-STATUS.md — 夜信 Night Letter

> 最后更新：2026-06-22

## 项目概述

「夜信」是一款面向年轻人和轻度心理疗愈人群的梦境记录与情绪对话 Web App。核心理念是"把梦写成信"，通过温暖的 UI 和 AI 对话帮助用户记录梦境、觉察情绪、获得陪伴。

## 技术栈

- **前端**：纯 HTML/CSS/JS 单文件应用（无框架依赖）
- **后端**：腾讯云 CloudBase（认证 + 数据库）
- **AI**：通义千问 qwen-turbo API（对话生成）
- **部署**：GitHub Pages（静态托管）
- **版本控制**：Git + GitHub

## 设计规范

- **色彩**：深蓝夜空主题（`#0d0d1f`），暖金强调色（`#d4b896`）
- **字体**：Noto Serif SC（标题）+ Noto Sans SC（正文）
- **动画**：极光背景（3 色块 + 12 粒子），低性能消耗
- **布局**：移动端优先，max-width 430px，9:16 比例
- **交互**：柔和过渡动画（cubic-bezier），触感反馈（scale 缩放）

## 已知问题

- 文件为 GBK/UTF-8 混合编码，Git merge 可能引入编码损坏
- 单文件架构（index.html ~200KB），后续可考虑拆分
- Qwen API 密钥硬编码在前端（仅用于演示）
- 云端同步无去重（重复登录会产生重复文档）— Sprint 2 待修
- 广场点赞无去重、评论者身份硬编码 — Sprint 3 待修
